## How to Use

1. Download the Excel file from this repository.
2. Open the file in Microsoft Excel.
3. Navigate to the Dashboard sheets:
   - Overall Dashboard
   - Brand Dashboard
   - Product Dashboard
4. Use the filters and pivot tables to explore the data.
5. Review the insights and visualizations for business analysis.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning
- Dashboard Creation
